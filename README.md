# 🌍 Visa Approval Prediction using Machine Learning  
### 🚀 Predicting visa certification outcomes with data-driven insights

---

## 📌 Overview  
This project focuses on building a machine learning model to predict whether a visa application will be **approved or denied** based on applicant and employer characteristics.  

The solution helps streamline the visa approval process and reduce manual effort.

---

## 🎯 Objective  
- Analyze factors influencing visa approval  
- Build a classification model to predict case status  
- Recommend profiles with higher chances of approval  

---

## 📂 Dataset Features  

- **case_id** – Unique application ID  
- **continent** – Applicant’s continent  
- **education_of_employee** – Education level  
- **has_job_experience** – Work experience (Y/N)  
- **requires_job_training** – Training requirement (Y/N)  
- **no_of_employees** – Company size  
- **yr_of_estab** – Company establishment year  
- **region_of_employment** – Job location in the US  
- **prevailing_wage** – Standard wage for the role  
- **unit_of_wage** – Wage unit (Hourly/Weekly/Monthly/Yearly)  
- **full_time_position** – Job type (Full-time/Part-time)  
- **case_status** – Target variable (Certified/Denied)  

---
